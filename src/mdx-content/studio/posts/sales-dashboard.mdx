# 📌 쇼핑몰 매출 분석 대시보드 (엑셀 + 웹 템플릿)

## 🧾 프로젝트 요약

쇼핑몰 운영자를 위한 매출 분석 대시보드를 제작했습니다.  
엑셀과 웹 버전을 함께 제공하여 **데이터 입력 → 자동 분석 → 의사결정**까지 이어지는 구조를 설계했습니다.

---

## 📂 프로젝트 개요

쇼핑몰 운영 시 발생하는  
매출, 광고비, 상품별 성과를 한눈에 파악할 수 있도록  
**엑셀 템플릿과 웹 대시보드를 함께 제작한 프로젝트**입니다.

단순 시각화가 아닌  
**운영 의사결정에 필요한 데이터 구조 설계**에 집중했습니다.

---

## 🎯 문제 정의

기존 쇼핑몰 운영자들의 공통 문제:

- 광고비 대비 효율 판단 어려움  
- 어떤 상품이 실제 수익을 내는지 파악 어려움  
- 매출은 증가하지만 순이익 구조 불명확  
- 데이터는 있지만 활용이 어려운 상태  

👉 **“데이터는 있는데 판단이 안 되는 문제” 해결**

---

## 💡 해결 방식

### 1. 엑셀 기반 자동화 템플릿

- 입력 시트에 값 입력 시 대시보드 자동 반영  
- 매출 / 광고비 / 순이익 자동 계산  
- 초보자도 바로 사용 가능하도록 설계  

---

### 2. 웹 기반 대시보드 (React / Next.js)

- 실시간 데이터 반영 구조  
- 차트 기반 시각화 (매출, 광고비, 상품별 성과)  
- 확장 가능한 컴포넌트 구조 설계  

---

### 3. 단일 HTML 버전 제작 (배포용)

- 별도 설치 없이 실행 가능  
- 진입장벽 최소화 (영업용 템플릿)  
- Chart.js 기반 인터랙션 구현  

---

## 🛠️ 사용 기술

- Excel (함수, 데이터 구조 설계)  
- HTML / CSS / JavaScript  
- Chart.js  
- React  
- Next.js  
- Tailwind CSS  
- Shadcn UI  

---

## 📊 주요 기능

- 일별 매출 추이 분석  
- 광고비 대비 매출 (ROAS) 자동 계산  
- 상품별 매출 순위  
- 총매출 / 순이익 자동 계산  
- KPI 카드 기반 요약 정보 제공  
- 데이터 입력 시 실시간 반영  

---

## 🚀 성과 및 특징

- 비개발자도 사용할 수 있는 구조 설계  
- 엑셀 + 웹 버전 이중 제공으로 활용성 확대  
- 무료 템플릿 → 맞춤 제작으로 이어지는 영업 구조 설계  
- 실제 쇼핑몰 운영 기준 데이터 흐름 반영  

---

## 💰 확장 방향 (비즈니스 모델)

- 무료 템플릿 배포 (리드 확보)  
- 맞춤형 대시보드 제작 (유료 전환)  
- 고객 데이터 구조에 맞춘 분석 환경 제공  

---

## 📩 한 줄 소개

> 쇼핑몰 운영 데이터를 “보여주는 것”이 아니라  
> **“결정할 수 있게 만드는” 대시보드**를 제작했습니다.
